About

coolTalk is an open-source, easy to integrate, privacy-centric & self-hostable messaging platform for your applications.

tip

You can check out the client side integration on this page (see right bottom corner) ðŸ‘‰ðŸ‘‡

Privacy-centric#

We do NOT collect any user data. Most chat application collects a LOT of things, like IP address, location information, etc...

We DON'T.

If something is collected, it first needs to be approved by the user.

Easy integration#

You can integrate it with one line. It works with any framework (React/Vue/Svelte) or framework-less environment.

Self-hostable#

If you want, you can host it for yourself with all the features, no restrictions, but you need to take care of the server for yourself!

Hassle-free installation and upgrade#

Install and upgrade your own self-hosted instance with one button!
