Conversations - Admin part

Conversations are automatically gathered and shown on the dashboard. The main two statues of a conversation are:

  • Unread: It means there is a new chat message from the user which an administrator does not yet answer.
  • Waiting: It means the administrator answered the user message, but the conversation is still not solved.

You can change the status of a conversation manually to unread or solved. Select a conversation and click on one of the buttons:

Send email notifications#

You can manually send email notifications in case the user added it's email address or you set manually with the correct data attribute. Counter shows the number of emails sent.

Email contains the full chat history of the conversation.
